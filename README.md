Stock Trading Spring Boot Application

1. Developed a secure multi-user trading platform using Java, Spring Boot, Spring MVC, and JPA/Hibernate following layered architecture principles.
2. Implemented Spring Security with BCrypt encryption for authentication, authorization, and session management.
3. Designed transactional trade execution workflow using @Transactional to ensure data consistency and integrity.
4. Built RESTful controllers and service-layer business logic for portfolio management, P&L calculation, and order processing.
5. Integrated relational database using Hibernate ORM with optimized entity mapping and repository abstraction.
6. Applied exception handling, input validation, and dependency injection to maintain scalability, maintainability, and clean architecture standards.
